**1. Create `src/huntOverviewPanel.ts`:**

Follow the exact structural pattern of `src/drainViewer.ts` (DrainTemplatePanel). Key differences: no queryId (singleton panel), no pin state, adds session diff logic.

Type for stored session hashes:
```typescript
type ArtifactHashMap = Record<string, string>;  // artifactId -> contentHash
```

**`computeArtifactHashes(store: HuntDataStore): ArtifactHashMap`** (module-level helper):
- Call `store.getQueries()` and `store.getReceipts()`. For each loaded entry, add `id -> data.contentHash` to the map.
- Call `store.getHunt()`. If mission loaded, add `"MISSION" -> data.signal` (use signal as a simple change indicator). If hypotheses loaded, add `"HYPOTHESES" -> JSON.stringify(data.active.map(h => h.status))` (verdict changes). If state loaded, add `"STATE" -> data.lastActivity`.
- Return the map.

**`computeSessionDiff(previous: ArtifactHashMap, current: ArtifactHashMap): SessionDiff`** (module-level helper):
- For each key in `current` not in `previous`: `{ artifactType: inferType(key), artifactId: key, diffKind: 'added', timestamp: new Date().toISOString() }`.
- For each key in both where hash differs: `{ ..., diffKind: 'modified', ... }`.
- For each key in `previous` not in `current`: `{ ..., diffKind: 'removed', ... }`.
- `inferType(id)`: if starts with "QRY" -> "query", "RCT" -> "receipt", "HYP" -> "hypothesis", otherwise lowercase of the id.
- Build summary string: `"N added, M modified, K removed since last session"` (omit zero categories).
- Return `{ entries, summary }`.

**`getDiagnosticsHealth(): { warnings: number; errors: number }`** (module-level helper):
- Call `vscode.languages.getDiagnostics()`. Filter to diagnostics with `source === 'THRUNT Evidence'`. Count by severity: `Warning` -> warnings, `Error` -> errors.

store.initialScanComplete().then(() => {
  const previousHashes = context.workspaceState.get<Record<string, string>>(SESSION_HASH_KEY, {});
