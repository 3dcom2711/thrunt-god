The reason for inlining rather than changing the npm script: we want the lcov reporter only in CI (local dev doesn't need lcov files generated on every run). Use this exact step:

```yaml
      - name: Run tests with coverage
        shell: bash
        run: |
          npx c8 --check-coverage --lines 70 \
            --reporter text --reporter lcov \
            --include 'thrunt-god/bin/lib/*.cjs' \
            --exclude 'tests/**' --all \
            node scripts/run-tests.cjs
```

3. **Add coverage artifact upload step**: After the coverage step, add an upload step that runs ONLY on ubuntu-latest AND Node 22 (the primary LTS). Use this exact step:

```yaml
      - name: Upload coverage report

        uses: actions/upload-artifact@v4
        with:
          name: coverage-report
          path: coverage/lcov.info
          retention-days: 30
```
