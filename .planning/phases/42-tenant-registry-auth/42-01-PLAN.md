**Imports:** `{ loadConfig, planningDir, planningPaths, output, error }` from `./core.cjs`, `{ setConfigValue }` from `./config.cjs`. Lazy-require `./runtime.cjs` inside async functions that need `assessConnectorReadiness` (avoid circular deps).

**Zod Schema (TenantConfigSchema):**
Use `require('zod')` (already a project dependency -- check node_modules). If Zod is NOT available, implement manual validation matching the same shape. The schema:
- `id`: string, regex `/^[a-z0-9][a-z0-9-]*$/`, max 64 chars
- `display_name`: optional string
- `tags`: optional array of strings, default `[]`
- `enabled`: boolean, default `true`
- `connectors`: record of connector_id (string) to `{ profile: string (required), parameters: object (optional) }`, must have at least 1 entry

**validateTenantConfig(tenant, config):**
- Validates against TenantConfigSchema (or manual checks if no Zod)
- For each `connectors[connectorId].profile`, verify it exists in `config.connector_profiles[connectorId][profileName]`
- Cross-check credential isolation: warn if two tenants in `config.tenants` reference the same env var value in different secret_refs (iterate all profiles referenced by all tenants, collect env var values, flag duplicates across tenants)
- Returns `{ valid: boolean, errors: string[], warnings: string[] }`

**assessTenantReadiness(tenantId, config, options = {}):**
- Look up `config.tenants[tenantId]`; if not found return `{ tenant_id, status: 'not_found', connectors: [] }`
- For each connector in `tenant.connectors`, call `runtime.assessConnectorReadiness(connectorId, config, { ...options, profile: connectorConfig.profile })`
- Aggregate into overall status: `'ready'` if all connectors readiness_status is ready/live_verified, `'partial'` if some are ready, `'unconfigured'` if none are ready
- Returns `{ tenant_id, display_name, enabled, status, connectors: [ { connector_id, profile, readiness_status, readiness_score, ... } ] }`

**cmdTenantList(cwd, raw):**
- `loadConfig(cwd)`, read `config.tenants || {}`
- For each tenant: `{ id, display_name, enabled, connector_count: Object.keys(t.connectors).length, connectors: Object.keys(t.connectors) }`
- `output({ tenants: [...], count: N }, raw)`

**cmdTenantStatus(cwd, tenantId, raw):**
- `loadConfig(cwd)`, verify tenant exists, call `assessTenantReadiness(tenantId, config, { cwd })`
- `output(readinessResult, raw)`

**cmdTenantAdd(cwd, args, raw):**
- Parse args: first non-flag arg is tenant ID, `--display-name "..."`, `--connector sentinel:profile-name` (repeatable), `--tag healthcare` (repeatable)
- Validate tenant ID format (regex + max 64)
- Build tenant object: `{ display_name, tags, enabled: true, connectors: { [connectorId]: { profile: profileName } } }`
- Run `validateTenantConfig(tenant, config)` -- if errors, `error(...)` and exit
- `setConfigValue(cwd, 'tenants.' + tenantId, tenant)`
- `output({ added: true, tenant_id: tenantId, tenant }, raw)`

**cmdTenantDisable(cwd, tenantId, raw):**
- `loadConfig(cwd)`, verify tenant exists in `config.tenants`
- `setConfigValue(cwd, 'tenants.' + tenantId + '.enabled', false)`
- `output({ disabled: true, tenant_id: tenantId }, raw)`

**cmdTenantEnable(cwd, tenantId, raw):**
- `loadConfig(cwd)`, verify tenant exists in `config.tenants`
- `setConfigValue(cwd, 'tenants.' + tenantId + '.enabled', true)`
- `output({ enabled: true, tenant_id: tenantId }, raw)`

**cmdTenantDoctor(cwd, args, raw):**
- `loadConfig(cwd)`, iterate all tenants, call `assessTenantReadiness` for each
- `output({ tenants: results, summary: { total, ready, partial, unconfigured } }, raw)`

**module.exports:** Export all 8 functions: `validateTenantConfig`, `assessTenantReadiness`, `cmdTenantList`, `cmdTenantStatus`, `cmdTenantAdd`, `cmdTenantDisable`, `cmdTenantEnable`, `cmdTenantDoctor`.
