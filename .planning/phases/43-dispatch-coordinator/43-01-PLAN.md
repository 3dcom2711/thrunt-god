  <behavior>
    resolveTenantTargets:
    - Given config with 3 tenants, options.tags=["healthcare"] returns only tenants with that tag
    - Given options.connector_id="sentinel" returns only tenants with sentinel connector configured
    - Given options.tenant_ids=["acme-corp"] returns only that tenant
    - Disabled tenants excluded by default; options.exclude_disabled=false includes them
    - Returns array of {tenant_id, connector_id, profile_name, parameters, display_name, tags}
    - When connector_id specified, each result has exactly that connector_id; when not specified, returns one target per connector per tenant

    cloneTenantSpec:
    - Takes baseSpec and target, returns new QuerySpec via createQuerySpec()
    - connector.id = target.connector_id, connector.profile = target.profile_name, connector.tenant = target.tenant_id
    - parameters = { ...baseSpec.parameters, ...target.parameters } (tenant overrides base)
    - evidence.tags includes "tenant:{target.tenant_id}"

    dispatchMultiTenant:
    - Executes N targets with concurrency semaphore (default 5, configurable)
    - Promise.allSettled pattern: one rejection does not affect others
    - Per-tenant token_cache: each executeTenantQuery receives fresh new Map()
    - Per-tenant timeout: spec.execution.timeout_ms applies per-tenant, not globally
    - Global timeout (default 600s, config dispatch.global_timeout_ms): cancels remaining and marks as timeout
    - Returns MultiTenantResult with dispatch_id (MTD-{YYYYMMDDHHMMSS}-{RANDOM8}), summary, tenant_results[], errors[]
    - Summary has tenants_targeted, tenants_succeeded, tenants_partial, tenants_failed, total_events, total_entities, wall_clock_ms
    - Each tenant_result has tenant_id, display_name, status (ok|partial|error|timeout|skipped), envelope, artifacts, timing
  </behavior>

    2. **Create dispatch.cjs** in thrunt-god/bin/lib/ with:

       require runtime.cjs for: createQuerySpec, executeQuerySpec, toArray, nowIso, makeId, createBuiltInConnectorRegistry, resolveConnectorProfile, resolveSecretRefs, createResultEnvelope.
       require core.cjs for: loadConfig.

       **resolveTenantTargets(config, options = {})**
       - Read config.tenants object
       - Filter by options.tenant_ids (if provided, only those IDs)
       - Filter by options.exclude_disabled !== false (exclude disabled tenants)
       - Filter by options.tags (tag intersection -- tenant must have ALL specified tags)
       - Filter/expand by options.connector_id (if provided, only tenants with that connector; each matching tenant produces one target for that connector. If not provided, each tenant produces one target per connector entry)
       - Return array of { tenant_id, connector_id, profile_name, parameters, display_name, tags }

       **cloneTenantSpec(baseSpec, target)**
       - Call createQuerySpec with spread of baseSpec fields but overriding:
         connector: { ...baseSpec.connector, id: target.connector_id, profile: target.profile_name, tenant: target.tenant_id }
         parameters: { ...baseSpec.parameters, ...(target.parameters || {}) }
         evidence: { ...baseSpec.evidence, tags: [...toArray(baseSpec.evidence?.tags), `tenant:${target.tenant_id}`] }
       - Return the new spec

       **executeTenantQuery(tenantSpec, target, registry, config, options)**
       - Internal helper (not exported)
       - Create a fresh token_cache = new Map() for this tenant
       - Call executeQuerySpec(tenantSpec, registry, { cwd: options.cwd, config, token_cache, artifacts: { tenant_id: target.tenant_id } })
       - Return { tenant_id: target.tenant_id, display_name: target.display_name, status, envelope, artifacts, timing }
       - Derive status from envelope: 'ok' if no errors, 'partial' if warnings but has events, 'error' if errors array non-empty

       **dispatchMultiTenant(baseSpec, targets, registry, config, options = {})**
       - concurrency = options.concurrency || config?.dispatch?.concurrency || 5
       - globalTimeoutMs = options.global_timeout_ms || config?.dispatch?.global_timeout_ms || 600_000
       - dispatch_id = `MTD-${timestamp}-${random8}` using makeId-like pattern
       - startedAt = nowIso()
       - Use manual semaphore loop (not Promise.all):
         pending = [...targets], active = new Set(), results = []
         While pending.length > 0 || active.size > 0:
           Fill active up to concurrency from pending (clone spec, launch executeTenantQuery wrapped in .then/.catch/.finally)
           If active.size > 0: await Promise.race(active), push completed to results
         Global timeout: wrap the whole dispatch loop in a Promise.race against a setTimeout. If global timeout fires, mark all remaining pending as {status:'timeout', tenant_id}.
       - Build summary: count statuses, sum events/entities, compute wall_clock_ms
       - Return MultiTenantResult: { version: '1.0', dispatch_id, summary, tenant_results, errors }

       **Exports:** resolveTenantTargets, cloneTenantSpec, dispatchMultiTenant
