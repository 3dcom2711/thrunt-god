    **Step 2: Create detections.cjs**

    Create `mcp-hunt-intel/lib/detections.cjs` with the following exports:

    1. **parseSigmaRule(yamlText, filePath)** -- Parse Sigma YAML. Use `require('js-yaml').load()`. Extract technique IDs from `tags` array filtering by `/^attack\.t\d{4}/i`, strip `attack.` prefix, uppercase. Extract tactics from tags filtering by `/^attack\.[a-z]/` excluding technique pattern, title-case. Store `falsepositives`, `author`, `status`, `date`, `modified`, `references`, `related` in metadata JSON. Return null for malformed/missing-title. Log warnings to stderr.

    2. **parseEscuRule(yamlText, filePath)** -- Parse ESCU YAML. Use `require('js-yaml').load()`. ID from `doc.id`, title from `doc.name`, technique_ids from `tags.mitre_attack_id`, query from `doc.search`, logsource from `doc.data_source` (joined), severity empty string (ESCU uses risk_score instead). Store `analytic_story`, `asset_type`, `security_domain`, `product`, `data_models`, `risk_score` (from `rba.risk_objects[0].score`), `known_false_positives`, `how_to_implement`, `references`, `type`, `status` in metadata JSON. Return null for malformed/missing-name.

    3. **parseElasticRule(tomlText, filePath)** -- Parse Elastic TOML. Use `require('smol-toml').parse()`. Iterate ALL `rule.threat` entries (array of tables), extracting `tactic.name`, `technique.id`, and `technique.subtechnique[].id`. Deduplicate. Store `maturity`, `creation_date`, `updated_date`, `integration`, `risk_score`, `rule_type` (from `rule.type`), `language`, `tags`, `references`, `license` in metadata JSON. Return null for malformed/missing-name.

    4. **parseKqlRule(mdText, filePath)** -- Parse KQL markdown. Extract title from first H1/H2 heading, falling back to filename (basename sans .md extension). Extract KQL from fenced code blocks tagged `kql`, `kusto`, or `csl`. If none found, try generic fenced blocks with KQL heuristic (contains `where|project|summarize|extend|DeviceEvents|DeviceProcessEvents`). Return null if no code blocks found. Extract technique IDs via `/\bT\d{4}(?:\.\d{3})?\b/g` regex on full text. Extract Microsoft table names via regex covering DeviceEvents, DeviceProcessEvents, DeviceNetworkEvents, DeviceFileEvents, DeviceRegistryEvents, DeviceImageLoadEvents, DeviceLogonEvents, IdentityDirectoryEvents, IdentityLogonEvents, IdentityQueryEvents, EmailEvents, EmailAttachmentInfo, CloudAppEvents, AADSignInEventsBeta, AlertEvidence, AlertInfo, SecurityEvent, Syslog, SigninLogs, AuditLogs. ID from `kql:` + relative path from dir root (use full relative path, not just filename, to avoid collisions). Store `tables` and `code_block_count` in metadata JSON. Description is first 500 chars of markdown with code blocks stripped.

    5. **ensureDetectionsSchema(db)** -- Create `detections` table (id TEXT PRIMARY KEY, title TEXT NOT NULL, source_format TEXT NOT NULL, technique_ids TEXT, tactics TEXT, severity TEXT, logsource TEXT, query TEXT, description TEXT, metadata TEXT, file_path TEXT) + `detections_fts` USING fts5(title, description, query, technique_ids, tokenize='porter unicode61') + indexes idx_det_source on source_format and idx_det_severity on severity. All `CREATE ... IF NOT EXISTS`.

    6. **insertDetection(db, row)** -- Prepared statement INSERT OR IGNORE into detections for all 11 columns. If `result.changes > 0`, also INSERT into detections_fts (title, description, query, technique_ids). Return `result.changes`.

    7. **indexSigmaDirectory(db, dirPath)** -- `fs.readdirSync(dirPath, { recursive: true })`, filter `.yml`/`.yaml`, read each, call parseSigmaRule, call insertDetection. Skip nulls and empty IDs (`sigma:`). Return count of indexed rules. Wrap directory read in try/catch, warn to stderr on failure.

    8. **indexEscuDirectory(db, dirPath)** -- Same pattern as indexSigmaDirectory but calls parseEscuRule, filters `.yml`/`.yaml`.

    9. **indexElasticDirectory(db, dirPath)** -- Same pattern but calls parseElasticRule, filters `.toml`.

    10. **indexKqlDirectory(db, dirPath)** -- Same pattern but calls parseKqlRule, filters `.md`. Pass relative path (not just filename) as filePath to parseKqlRule for ID generation.

    11. **searchDetections(db, query, opts)** -- FTS5 search on detections_fts MATCH with BM25 ranking. JOIN back to detections table for full row. Support opts: `source_format` (WHERE filter), `severity` (WHERE filter), `technique_id` (LIKE on technique_ids column), `limit` (default 20). Wrap in try/catch, return empty array on malformed query. Use same pattern as `searchTechniques` in intel.cjs.

    12. **populateDetectionsIfEmpty(db)** -- Check `SELECT COUNT(*) FROM detections`. If 0: index bundled sigma-core from `path.join(__dirname, '..', 'data', 'sigma-core', 'rules')`, then call `indexEnvPaths(db, 'SIGMA_PATHS', indexSigmaDirectory)`, `indexEnvPaths(db, 'SPLUNK_PATHS', indexEscuDirectory)`, `indexEnvPaths(db, 'ELASTIC_PATHS', indexElasticDirectory)`. Helper `indexEnvPaths(db, envKey, indexFn)` splits env var by `path.delimiter`, calls indexFn for each existing directory. Wrap in `db.transaction(() => {...}).immediate()`.

    All exports at bottom of file via `module.exports = { ... }`.
